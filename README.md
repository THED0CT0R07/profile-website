# Aaron Kerbs Portfolio Website

## Purpose of the Repository

This repository contains my personal portfolio website created as part of my software development coursework and professional portfolio development.

The website was built using HTML, CSS, and JavaScript and is designed to showcase:
- My background and skills as a software development student
- Future projects and portfolio work
- Sample website design concepts and pricing pages
- A responsive and modern user interface

The website includes:
- Responsive mobile navigation with a hamburger menu
- Multiple design pages
- Contact form section
- Professional gradient styling
- GitHub and portfolio links
